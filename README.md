# Wishlist App on React

## Available features

### `Add a wish`

Adds new item in the wishlist that can be changed to finished by clicking on it

### `Edit a wish`

You can simply edit a wish by clicking on a button named "edit", and change its value

### `Delete a wish`

Simply deletes an item

### `Filter by Finished`

This function filters the wishlist and puts finished wishes first and unfinished right after finished ones

## LocalStorage

Everything is stored in a local storege.

## The modest design ever

Design supposed to be powered by material ui or Ant design, but due to lack of time and version mismatch. It was decided to keep it in vanilla style)
